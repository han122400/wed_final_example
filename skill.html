HTML

웹 페이지의 구조를 설계하는 마크업 언어입니다. 각 요소의 배치와 의미를 정의합니다.

Home, Team, Skill, MyPage, Login, Singup 페이지를 구현하는데 사용되었습니다.

CSS

HTML로 만든 구조에 디자인을 입히는 스타일 언어입니다. 색상, 레이아웃, 애니메이션 등을 담당합니다.

전체적인 웹페이지의 스타일을 정의하는데 사용되었으며, 각 버튼들의 디자인, 호버 기능을 통한 스타일 변화,
반응형 웹 디자인 구성 등 다양한 시각적 요소들을 구현하는데 사용되었습니다.

JavaScript

웹 페이지에 동적인 기능을 추가하는 프로그래밍 언어입니다. 사용자와의 상호작용을 구현합니다.

localstorage와 sessionstorage를 이용한 로그인 구현,
event 리스너와 onclick 기능을 통해 버튼을 클릭 할시 나오는 각 버튼들의 상호작용 기능,
메인화면에서 텍스트가 타이핑 되듯이 나오는 typewriter 효과, To-Do 리스트 구현을 위해 사용 된 Date 객체와
setTimeout, setInterval 등 다양한 내장 함수를 활용하여 동적인 UI를 구현하는데 사용되었습니다.

ChatGPT

자연어 처리 AI로, 코드 설명, 오류 해결, 아이디어 제안 등 다양한 개발 지원을 제공합니다.

Github

코드 호스팅 플랫폼으로, 버전 관리, 협업, 코드 리뷰 등 다양한 개발 지원 기능을 제공합니다.

Notion

협업 및 노트-taking 도구로, 프로젝트 관리, 아이디어 정리, 문서화 등을 지원합니다.

Discord

커뮤니케이션 플랫폼으로, 팀원 간의 실시간 소통, 파일 공유, 음성 채팅 등을 지원합니다.
